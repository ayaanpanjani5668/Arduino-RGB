# Arduino-RGB
This project allows users to control an RGB LED using an Arduino by entering color names through the Serial Monitor. Depending on the input, the Arduino changes the RGB LED to display different colors such as red, green, blue, yellow, aqua, and purple.
Features:
- Control an RGB LED through Serial Monitor commands
- Display multiple colors by mixing red, green, and blue LEDs
- Simple and interactive project for learning Arduino outputs and color mixing
- Demonstrates the use of digital pins and user input through serial communication
Components Used:
- Arduino Uno
- RGB LED (Common Cathode or Common Anode)
- 330Ω resistors (3)
- Breadboard
- Jumper wires
- USB cable
